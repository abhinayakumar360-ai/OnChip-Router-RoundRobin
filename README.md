# On-Chip Router using Round Robin Arbitration

## Description
This project implements an on-chip router using Round Robin Arbitration in Verilog HDL.

## Features
- Fair arbitration using round robin scheduling
- Multiple input and output ports
- Efficient data packet routing

## Concepts Used
- RTL Design
- Arbitration Techniques
- Digital System Design

## Tools Used
- Verilog HDL
- Xilinx

## Applications
- Network on Chip (NoC)
- Communication systems
